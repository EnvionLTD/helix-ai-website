# Helix AI — Website

Static front-end site for Helix AI. Pure HTML/CSS, no build step, no dependencies.

## Pages
- `index.html` — Home
- `services.html` — Services
- `how-it-works.html` — Workflow + process
- `about.html` — About
- `contact.html` — Contact (mailto + tel links direct)

## Deploy to Vercel
1. Push this repo to GitHub.
2. Import the repo in Vercel.
3. Framework preset: **Other** (or leave blank — it's static).
4. Build command: leave empty.
5. Output directory: `./`
6. Deploy.

## Local preview
Open `index.html` in a browser. That's it.
